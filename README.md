Descrição do jogo

Este projeto consiste em uma recriação do jogo Forager, desenvolvida em JavaScript puro. O jogo possui visão top down e foco em coleta de recursos, gerenciamento de inventário, sistema de crafting e progressão de nível. O jogador controla um ratinho que explora o mapa em busca de recursos para evoluir.

Mecânicas principais

Coleta de recursos ao andar sobre os itens
Inventário para visualização dos recursos coletados
Sistema de crafting para combinação de itens
Progressão de nível conforme o avanço do jogador
Sistema de spawn de novos recursos no mapa

Controles

WASD ou setas – movimentação do personagem
C – abrir menu de crafting
Mouse – clicar nos botões de crafting
Espaço – ataque

Como executar

Extraia o arquivo .zip do projeto
Abra o arquivo index.html em um navegador
Utilize WASD ou as setas para se movimentar
Ande sobre os recursos para coletá-los
Pressione C para acessar o menu de crafting
Utilize o mouse para craftar itens

Tecnologias utilizadas

CSS3
